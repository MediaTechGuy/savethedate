# 💍 Blessing & Tolulope — Save the Date

An animated wedding invitation website hosted via GitHub Pages.

## 📁 File Structure

```
savethedate/
├── index.html        ← Main page
├── style.css         ← All styles & animations
├── script.js         ← Envelope open + music logic
├── assets/
│   └── music.mp3     ← ⬅ PUT YOUR SONG FILE HERE
└── README.md
```

## 🎵 Adding the Music

1. Download **"Can I Have This Dance"** by Vanessa Hudgens & Zac Efron (HSM 2008) as an MP3.
2. Rename it to `music.mp3`.
3. Place it inside the `assets/` folder.

If the MP3 file is missing or blocked by the browser, the site automatically falls back to streaming via YouTube.

## 🚀 Deploying to GitHub Pages

1. Push all files to your repository (`mediatechguy/savethedate`).
2. Go to **Settings → Pages**.
3. Set source to `main` branch, `/ (root)` folder.
4. Your site will be live at `https://mediatechguy.github.io/savethedate/`

## ✏️ Customisation

All design tokens (colours, fonts) are at the top of `style.css` under `:root { }`.  
Change `--gold` to any colour and the whole theme updates instantly.
